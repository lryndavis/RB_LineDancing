# Line Dancing Team App 

By Lauryn Davis + Patrick Sullivan 

A Ruby/Sinatra exercise in implementing one-to-many relationships. 

##### Technologies Used

* Ruby
* Sinatra
* Active Record
* PostgresQL
* rspec
* capybara
* HTML
* CSS
* Bootstrap

##### Setup

* Clone this repository
* Run Postgres
* In terminal:
* `bundle install`
* `rake db:create`
* `rake db:migrate`
* `rake db:test:prepare`
* `ruby app.rb`
